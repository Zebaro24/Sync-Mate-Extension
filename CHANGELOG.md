## v0.1.4 — 2026-06-27

## v0.1.3 — 2026-06-27

- Keep-alive WebSocket (~30с) + ресинк при возврате вкладки из фона — связь не рвётся в простое
- Устойчивость к вёрстке Rezka: ожидание `<video>`, try/catch на входе, ленивые селекторы — мелкая правка сайта больше не роняет расширение молча
- Синхронная смена фильма ведущим (loop-safe) + проверка живой комнаты по invite-ссылке + mutex записи комнаты
- Снятие оверлея-спиннера и блокировки пробела при смене эпизода/дисконнекте
- Popup: корректный индикатор синхронизации, фикс кнопки в модалке ника, нет вылета в ошибку при сбое обновления, работа в нескольких вкладках/комнатах
- Меньше нагрузки: убраны пер-кадровые логи (вырезаются в прод-сборке), троттлинг `status` до ~1/с
- Повтор подключения к той же комнате после ошибки, безопасное копирование ссылки, фолбэк хранилища для старого Firefox
- Панель участников: исчезновение «призраков» ушедших по TTL

## v0.1.2 — 2026-06-27

- Починены залипания эхо-флагов плеера (play/seek/waiting) — меньше рассинхрона
- Авто-реконнект WebSocket с честным статусом и ресинком после обрыва
- Отдача длительности ролика (`duration`) для готовности к концу видео
- Корректный порядок очистки подписок при закрытии
- Навигация по `set_video` только на `https://rezka.ag/`
